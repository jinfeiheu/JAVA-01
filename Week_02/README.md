学习笔记

- 不配置 xms,初始内存大约是堆内存的1/64
- 打开自适应后，配置的参数会根据初始值，发生自适应的变化。     
- 总体效果：指定xms + 不开自适应 （约等于） 指定 xms  + 开自适应 （略大于） 不指定xms + 开自适应  （远大于） 不指定xms + 不开自适应
    
 >  以上是在使用机器1/8内存情况下的粗略测试，实际使用时，堆扩容和堆大小分别影响 应用线程的CPU时间和GC的时间，开自适应的好处还是很大的，指定xms也是有用的，减少了扩容时间。
